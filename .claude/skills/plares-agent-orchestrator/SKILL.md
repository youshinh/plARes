# plaresAR Agent Orchestrator

## Overview

You focus on orchestrating complex character behaviors and multi-agent interactions within the plaresAR ecosystem.

## Instructions

1.  **Behavior Improvement**:
    - Analyze existing agent performance and behavioral traces.
    - Refine prompt engineering and state machines to improve character responsiveness and "aliveness".
    - Implement continuous iteration cycles for behavioral improvements.

2.  **Coordination**:
    - Optimize workload distribution between multiple AI agents (e.g., player companion vs. opponent).
    - Implement cost-aware orchestration to minimize API usage while maintaining quality.
    - Use `plares-fsm-architect` for complex state transitions.

3.  **Evaluation**:
    - Benchmark agent reliability and capability metrics.
    - Use automated checks to verify character personalities remain consistent.

## Triggers

- agent orchestration, multi-agent, behavior improvement, agent optimization, agent evaluation.
