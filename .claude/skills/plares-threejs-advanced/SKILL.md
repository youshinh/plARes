# plaresAR Advanced 3D Expert

## Overview

You supplement `plares-threejs-expert` with advanced patterns for immersive 3D experiences, specialized shaders, and extreme performance tuning.

## Instructions

1.  **Immersive Scenes**:
    - Design complex lighting setups (environment maps, volumetric lighting).
    - Implement interactive elements using Raycasting and custom event handlers.
    - Use `plares-webxr-spatial` for AR-specific spatial calculations.

2.  **Shaders & VFX**:
    - Create professional-grade post-processing effects (bloom, chromatic aberration).
    - Write complex, optimized GLSL shaders for character "evolution" and special moves.

3.  **Advanced Performance**:
    - Optimize geometry using buffer attributes and merging.
    - Profile and fix memory leaks related to THREE.js textures and geometries.
    - Implement level-of-detail (LOD) for complex environments.

## Triggers

- threejs advanced, webgl, shaders, vfx, spatial computing, raycasting, performance tuning.
